SlackMud

What is a MUD?

  • Originally from late 70s..
  • .. original papers / implementation at University of Essex
  • Text based gameplay
  • Roleplay / puzzle based
  • The grandparent of modern day MMOs

SlackMud - Features

  • Fully configurable game system - using JSON files
    • Commands
    • Scripts
    • Objects
    • Locations
    • Skills
  • Built to "just work" when deployed to a webserver
  • Has built in web socket and bot framework linking
  • Easy to set up to use Azure hosting (for free)
